|-------|-------|

| Earrings | +2 Str/Dex/Int |

### PvP Formulas

| Mechanic | Formula |
|----------|---------|
| Hit Chance | `0.69 × ((Skill + Tactics×2) / 300)` |
| AR Absorption | `Random(AR/4.3 to AR/2.4)` |

## For AI Assistants

When implementing a feature:
1. **First**: Check INDEX.md for the topic
2. **Second**: Read the authoritative section in `Core/DESIGN_DECISIONS.md`
3. **Third**: Read the details document in `Systems/` if one exists
4. **Fourth**: Check ROADMAP.md for implementation status

**If documents conflict**: DESIGN_DECISIONS.md wins. Always.
